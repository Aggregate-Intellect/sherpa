Sherpa AI API Reference
=======================

.. image:: https://img.shields.io/badge/Sherpa-AI-blue
   :alt: Sherpa AI

Welcome to the Sherpa AI API reference documentation. This comprehensive guide describes all modules and classes available in the Sherpa AI framework.

Sherpa AI is an advanced AI agent framework that provides:

* **Multi-agent orchestration** for complex problem-solving
* **Action planning** for strategic execution of tasks  
* **Knowledge memory** systems for persistence across sessions
* **Specialized agents** for domain-specific tasks
* **Robust error handling** for production environments

The documentation is organized by modules, each containing detailed class and function references with examples.
