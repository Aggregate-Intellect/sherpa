## Quick Install
With pip:
```bash
pip install sherpa-ai
```

From source (with [poetry](https://python-poetry.org/)):
```bash
git clone https://github.com/Aggregate-Intellect/sherpa
cd sherpa/src
poetry install
```

From source (with pip editable mode):
```bash
git clone
cd sherpa/src
pip install -e .
```
